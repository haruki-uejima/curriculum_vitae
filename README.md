【発揮した成果】 実務でのインフラ業務、AWSを使うことは初めてでしたが、独学でAWS資格を取得した際に得た知識および業務時間内外に書籍やネット等で学習してキャッチアップしたことで、なんとかインフラのコード化とデプロイの自動化を実現することができました。
